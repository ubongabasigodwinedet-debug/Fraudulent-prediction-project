# Fraudulent-prediction-project
Fraudulent
## Business Challenge 
- Increasing digital transactions have led to a rise in fraudulent activities.
- Fraud can cause financial losses, reputational damage, and loss of customer trust.
- Businesses need systems that can quickly detect suspicious transactions.
- The challenge is to build a predictive model that distinguishes fraudulent transactions from legitimate ones.
- The model should improve fraud detection accuracy and reduce false alerts.

## Identify the target variable for the prediction
- Fraudulent: target variable
- 0 = Legitimate transaction
- 1 = Fraudulent transaction
